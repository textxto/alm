## Sobre o ALM

O **ALM – Luxury Design System** é a identidade visual e estrutural do Laboratório de Palavras. Um ecossistema editorial digital desenhado para transmitir elegância silenciosa, rigor técnico e clareza absoluta.

---

## Tecnologias

- **UIkit 3.21.3** – Framework CSS
- **Google Fonts** – Inter e Cormorant
- **CSS Grid + Flexbox** – Layout responsivo
- **JavaScript (vanilla)** – Interações e tema

---

## Funcionalidades

- Scroll horizontal com snap
- Barra de progresso
- Modo claro/escuro automático
- Navegação dupla (dots + lateral)
- Modo foco
- Micro-interações premium
- Acessibilidade (ARIA, teclado)
