## Direção para Ícones e Ilustrações Futuras

Para garantir a coesão e o padrão de luxo do ALM, qualquer ícone ou ilustração adicionada ao sistema **deve** obedecer às seguintes diretrizes:

### Para Ícones SVG

1.  **Traço (`stroke`)**: Utilizar `stroke-width: 1.5`. O traço deve ser limpo, sem quebras.
2.  **Preenchimento (`fill`)**: Manter `fill="none"`, a menos que o ícone represente um elemento sólido (ex: um círculo para uma fotografia).
3.  **Cor**: Herdar a cor do texto circundante (`currentColor`) ou utilizar as variáveis `--accent-primary` (azul) ou `--accent-secondary` (ouro) para destavery específicos.
4.  **Geometria**: Preferir formas geométricas simples e simétricas. Evitar curvas complexas e orgânicas.

### Para Ilustrações

1.  **Estilo**: Traço contínuo, minimalista, com espaços em branco generosos (estilo "respiração editorial").
2.  **Paleta**: Restrita à paleta principal do ALM: tons de azul (`#0055A4`), ouro (`#C6A43F`) e os cinzas estruturais (`#1E1E24`, `#6B7280`). O fundo deve ser transparente.
3.  **Função**: Devem comunicar uma ideia de forma abstrata e elegante, não literal. Servem para criar atmosfera, não para explicar funcionalidades.

> **Nota para Colaboradores:** Ativos que não respeitem estas diretrizes não serão integrados no sistema principal.
